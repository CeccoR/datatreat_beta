Analysis

Drag the vertical lines to set the Tauc linear regression region (red) and the baseline (magenta), or press ✦ Suggest intervals to place them automatically from the absorption edge (second-derivative method). Within each interval the best fit is chosen by sliding a window (its size is the regression-window value) and minimising NRMSE/R², where NRMSE = RMSE / (ymax−ymin) of the window. Normalising by the y-range keeps the fit on the steep linear part instead of a flat low-value stretch that only has a small absolute RMSE, so it is markedly more stable. Eg is extracted from both the x-axis intersection and the baseline intersection of the regression line. The Tauc exponent is 0.5 for indirect semiconductors and 2 for direct semiconductors.

Instrumental standard

Select a measured standard (large, strain-free crystallites — LaB₆, Si…) to characterise the instrumental broadening β_instr(2θ). Once chosen, that sample is removed from the Analysis panel and analysed here on its own: it is always normalised to its highest peak (= 1). Its peaks and FWHM feed the Caglioti fit (FWHM² = U·tan²θ + V·tanθ + W) that supplies the instrumental correction β² = β_obs² − β_instr² used for the samples' crystallite sizes. No crystallite size is computed for the standard itself, so the size column is omitted from its peak table.

Analysis

Per-sample view. Raw pattern (grey), moving-average smoothing (blue), SNIP baseline (orange) and detected peaks (green). The SNIP (Statistics-sensitive Non-linear Iterative Peak-clipping) baseline iteratively clips each point downward toward the average of its two neighbours at decreasing window widths — a smooth background that follows the broad diffuse scattering without being pulled into the peaks. A larger window gives a slower-varying baseline. Peak search: a local maximum is accepted if it exceeds Peak min. height and its prominence (height above the highest valley between it and any taller neighbour) exceeds Peak min. prominence; candidates closer than Peak min. distance are merged keeping the tallest.

Crystallite size (Scherrer, classic): D = K·λ / (β·cosθ). β is measured classically on the smoothed, baseline-subtracted profile as the half-maximum full width (contains the whole Kα₁+Kα₂ doublet). K and λ are editable. Instrumental broadening: a chosen standard (large, strain-free crystallites) measures β_instr(2θ); the sample contribution is β² = β_obs² − β_instr². The Results chart shows size both without and (if a standard is set) with this correction. The Kα₂-deconvolved profile fit lives in the Fitting panel below.

Fitting

Cu Kα₂-deconvolved whole-pattern fit. The raw pattern (open circles) and the peaks inherited from the Analysis peak search are shown; press Fit sample/Fit all to fit a sum of Kα₁/Kα₂ doublets (Kα₁ + ½·Kα₂ at +Δ2θ) plus a polynomial background, all refined together (Levenberg–Marquardt). Profile, asymmetry and other hyperparameters are in Sample fit settings. By default each fit restarts from the SNIP baseline and the current peak search (manual peaks kept); a setting lets it instead continue from the last fit. A chosen standard (LaB₆…) is fitted with the FCJ axial-divergence kernel via Fit standard to calibrate the instrument geometry. The fitted doublet, its Kα₁/Kα₂ components and the residual (with Rwp) appear once a fit exists.

Sample fit — hyperparameters

Whole-pattern least-squares: Σ Kα₁/Kα₂ doublets + polynomial background, Levenberg–Marquardt. Multi-start over profile shape (5) × initial width 1–5× FWHM (5) = 25 starts, then a final refinement pass on all points. Used by Fit sample and Fit all.

Fixed instrumental geometry applied to samples. Refine these by running Fit standard on your LaB₆ (or other) standard — the calibrated value is written back here automatically.

Standard fit — instrument calibration

Fits the file chosen as Instrumental standard with axial-divergence (FCJ) asymmetry, refining a single geometry parameter G = S/L = H/L. Multi-start over shape × initial width (25 starts, each refining G) + a final all-points pass; expect ~1 min. The result is written back to the sample fit settings and reused on all samples.

Updated automatically after each standard fit (refined within [0.001, 0.1]).
